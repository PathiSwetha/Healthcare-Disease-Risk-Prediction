# 🏥 Healthcare Data Analysis & Disease Risk Prediction System

## 📌 Project Overview

The **Healthcare Data Analysis & Disease Risk Prediction System** is an end-to-end Machine Learning web application that analyzes healthcare data, identifies disease patterns, performs exploratory data analysis (EDA), and predicts heart disease risk using patient medical information.

The project combines **Data Science, Data Analysis, Machine Learning, and Flask Web Development** to provide an interactive healthcare analytics dashboard and disease prediction system.

---

## 🎯 Problem Statement

Healthcare organizations generate large volumes of patient data but often struggle to derive meaningful insights for disease prevention and treatment planning.

This project aims to analyze healthcare data and predict heart disease risk using Machine Learning techniques.
